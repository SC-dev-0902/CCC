## Scope
Fix the API endpoint and `getTestFilePath()` so that sub-stage identifiers like `11a` and `11a01` are passed through correctly instead of being stripped by `parseInt`.

---

## Background

The `GET /api/projects/:id/test-file-path?stage=N` endpoint currently does:
```js
const stageNumber = parseInt(req.query.stage, 10);
```

`parseInt("11a", 10)` returns `11` ‚Äî the letter suffix is silently dropped before it reaches `getTestFilePath()`. So requesting `?stage=11a` produces a path for `stage11`, not `stage11a`.

`getTestFilePath()` also assumes a numeric stage and uses `padStart(2, '0')`, which would produce wrong output for non-numeric input even if it reached the function.

Both must be fixed together.

## Task 1 ‚Äî Fix API endpoint in `server.js`

Read `server.js` and find `GET /api/projects/:id/test-file-path`. Locate the stage validation block (~line 568):

```js
const stageNumber = parseInt(req.query.stage, 10);
if (!stageNumber || stageNumber < 1) {
  return res.status(400).json({ error: 'stage query parameter is required (positive integer)' });
}
```

Replace with:
```js
const stageId = req.query.stage;
if (!stageId || !/^\d+[a-z]?\d*$/.test(stageId)) {
  return res.status(400).json({ error: 'stage query parameter is required (e.g. "11", "11a", "11a01")' });
}
```

Then update the call to `getTestFilePath()` immediately after ‚Äî change `stageNumber` to `stageId`:
```js
const relativePath = versions.getTestFilePath(found.project.name, stageId, activeVersion);
```

## Task 2 ‚Äî Fix `getTestFilePath()` in `src/versions.js`

Read the current implementation of `getTestFilePath()` (~line 316).

Replace the function body so it accepts a string stage identifier and pads only purely numeric values:

```js
function getTestFilePath(projectName, stageId, activeVersion) {
  const parts = activeVersion.split('.');
  const stageStr = /^\d+$/.test(String(stageId))
    ? String(stageId).padStart(2, '0')
    : String(stageId);
  const fileName = `${projectName}_test_stage${stageStr}.md`;
  if (parts.length === 3) {
    const majorMinor = `v${parts[0]}.${parts[1]}`;
    const full = `v${activeVersion}`;
    return path.join('docs', majorMinor, full, fileName);
  }
  return path.join('docs', `v${activeVersion}`, fileName);
}
```

Update the JSDoc comment above `getTestFilePath()` to replace `stageNumber` (integer) with `stageId` (string).
