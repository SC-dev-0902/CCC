## Background

CCC identifies test files using a regex. The current pattern only matches main-stage files:

```
_test_stage\d+\.md$
```

This means `CCC_test_stage11.md` is detected, but `CCC_test_stage11a.md` and `CCC_test_stage11a01.md` are invisible to CCC.

The fix is the same one-character change applied in two places in `src/versions.js`.

---

## Task 1 — Flat docs scan regex (~line 62)

Read `src/versions.js` in full before editing.

Find this line (inside the flat docs scan block):
```js
const testPattern = new RegExp(`^${escapedName}_test_stage\\d+\\.md$`);
```

Replace with:
```js
const testPattern = new RegExp(`^${escapedName}_test_stage\\d+[a-z]?\\d*\\.md$`);
```
